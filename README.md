## Description

This project consists of creating a custom C library called **libft**, which reimplements a set of standard C library functions as well as additional utility functions.

The main goal of this project is to deepen the understanding of fundamental programming concepts such as memory management, pointer manipulation, and string processing. By recreating these functions from scratch, it is possible to understand how they work internally rather than simply relying on existing implementations.

The library is divided into three main parts:

* **Libc functions**: Reimplementation of standard functions such as `strlen`, `memcpy`, and `atoi`.
* **Additional functions**: More advanced utilities for string manipulation like `ft_split`, `ft_strjoin`, and `ft_substr`.
* **Linked list functions**: Implementation of a basic linked list data structure and its associated operations.

All functions are written in compliance with the 42 coding standard and compiled into a static library (`libft.a`), which can be reused in future projects.

This project builds a strong foundation in C programming and prepares the student for more complex challenges involving low-level memory handling and data structures.
